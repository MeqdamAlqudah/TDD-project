# TDD-project
> In this project using TDD we wrote tests in order t build a class that functioned as our test expected

## Built With
- Ruby
- Rspec

## Getting Started
Using a linux terminal

Clone the project `git clone git@github.com:thecodechaser/TDD-project.git`

Then go into the projecct folder
`cd TDD-project`'

Run tests using the following command in your linux
terminal
`rspec spec`
